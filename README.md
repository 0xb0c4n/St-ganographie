# Stéganographie | 0xb0c4n & 0q3i

## Prérequis

- Python 3
- `pyfiglet` (pour ajouter des styles de texte à la console, si nécessaire)

## Installation

1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/0xb0c4n/Steganographie.git
   cd steganographie
   pip install -r requirements.txt
   ```
2. Exécutez le fichier :
   ```bash
   python3 main.py
   ```
